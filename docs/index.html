Project status

Independent and transparent.

R|X DC is an independent community project and is not affiliated with or endorsed by Xiaomi, Redmi, POCO, Samsung, Google or the scrcpy project. The current release is a desktop-companion MVP using permitted Android debugging tools; it is not yet an OEM-integrated Samsung DeX equivalent.
