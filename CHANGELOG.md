# Changelog

All notable changes will be documented in this file.

## [1.0.0] - 2026-07-11

### Added

- Electron/React desktop application with the R|X DC visual identity.
- Go loopback backend with per-launch authentication.
- USB and wireless ADB device discovery.
- Device information, screenshot preview, input controls, installed-app launch, file push/pull, and text injection.
- Optional scrcpy desktop window.
- Automated x64/ARM64 installer workflows for Windows, macOS, and Linux.
- Tag-driven GitHub Release publishing.
- Apache-2.0 licensing and project documentation.
